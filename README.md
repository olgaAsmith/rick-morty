## Getting Started

First, clone project:

```
git clone https://github.com/olgaAsmith/rick-morty.git
```
Go to main directory: 
```
cd rick-morty
```
Then install dependencies:

```bash
npm install
#or
yarn install
```
Finally, run dev mode: 

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
